# Hammerspoon Configs

## Installation

[Download here.](http://www.hammerspoon.org)

Or `brew cask install hammerspoon`.

## Usage

`hyper`: `Ctrl` + `Alt` + `Cmd`

- Maximize current window. `hyper` + `M`
- Center current window. `hyper` + `C`
- Rezide and center current window. `hyper` + `Shift` + `C`
